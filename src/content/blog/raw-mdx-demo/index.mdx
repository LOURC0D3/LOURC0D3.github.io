## 1. 여러 단락 들어간 Callout

`callout-warning` 매크로는 한 단락만 가능. mdx로 적으면 여러 단락 가능.

<Callout variant="warning" title="Affected versions">

다음 버전이 영향을 받습니다:

- 1.0.0 ~ 1.5.3
- 2.0.0 ~ 2.3.5

패치는 **2.3.6**, **1.5.4** 에 포함됨. 업그레이드 권장.

</Callout>

## 4. 연속된 Callout — 문제/풀이 구조

<Callout variant="problem">

</Callout>

## 7. mdx 블록 안 페이로드 escape 주의

<Callout variant="danger">

본문에 페이로드 그대로 적으면 깨집니다. 직접 escape 필요:

- `\{\{7*7\}\}` 처럼 적으면 JS expression으로 파싱되어 빌드 에러
- `\<script\>` 도 컴포넌트로 잘못 해석됨

안전한 방법:

```
{{7*7}}
<script>alert(1)</script>
```

→ 백틱이나 코드블록 안은 자동 안전.

본문에서 쓰려면 backslash escape: `\{\{7*7\}\}` → 화면엔 \{\{7\*7\}\} 로 나옴.

</Callout>
